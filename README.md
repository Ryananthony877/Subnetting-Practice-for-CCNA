CCNA Subnetting Practice Tools
A set of interactive browser-based tools to help study for the Cisco CCNA certification exam. No installation required — open any file in a browser and start practising.

Tools included
1. Full Subnetting Practice Tool
full_subnetting_practice_tool.html
The complete end-to-end subnetting drill. Given a random IP address and CIDR prefix, work through all five steps in order:

Step 1 — Subnet mask (clickable binary grid with auto-fill)
Step 2 — Magic number
Step 3 — Network address
Step 4 — Broadcast address
Step 5 — Usable hosts

Each step unlocks only after the previous one is answered correctly. A full subnet summary appears on completion. Each step has a collapsible "Show how to" hint if you get stuck.

2. Subnet Mask Practice Tool
subnet_mask_practice_tool.html
Focused drill on working out the subnet mask from a CIDR prefix. Click the bits in the binary grid to toggle them on and off — the mask octets auto-fill as you go. Covers all /1 through /32 prefixes, weighted towards CCNA exam favourites.

3. Subnetting Practice Tool
subnetting_practice_tool.html
Quick-fire drill covering the full set of subnet answers in one go — mask, network address, broadcast address, and usable hosts. Great for timed practice once the individual steps feel comfortable.

How to use

Download the HTML file(s)
Open in any browser (Chrome, Firefox, Safari)
No internet connection required


CCNA exam topics covered

Converting CIDR prefixes to subnet masks
The magic number (block size) method
Finding network and broadcast addresses
Calculating usable host ranges
/8, /16, /24 classful anchors
/25 through /30 exam favourites
/31 point-to-point and /32 host routes
